# BookPede

BookPede lets the users search Google Books for their favorite books. The users can then save the book to view it later or click on the "view book" link to navigate to Google Books.

## Technology Stack

- React
- MongoDB 
- Express.JS, Node.JS
- JavaScript
- NPM Packages: 
    1. React Router
    2. Axios 
    3. Mongoose
